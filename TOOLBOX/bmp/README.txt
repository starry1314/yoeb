BMP 圖片資料夾使用說明
===================================================================

此資料夾用於存放 POE 物品的圖片,供離線商店上架功能使用。

【如何使用】

1. 截取物品圖片:
   - 在遊戲中將需要上架的物品截圖
   - 建議只截取物品圖示部分 (約 30x30 像素)
   - 使用 Windows 內建的「剪取工具」或其他截圖軟體

2. 儲存為 BMP 格式:
   - 將截取的圖片儲存為 .bmp 格式
   - 檔名可以自訂 (例如: 混沌石.bmp, 崇高石.bmp)
   - 將所有 .bmp 檔案放入此資料夾

3. 設定價格 (重要!):
   - 編輯 prices.ini 檔案
   - 格式: 檔案名稱(不含.bmp)=價格
   - 範例:
     混沌石=1
     崇高石=150
     神聖石=80
   - 檔案名稱必須與 BMP 檔案完全一致(不含副檔名)

4. 啟動離線商店上架功能:
   - 按下 F6 啟動
   - 腳本會自動載入所有 .bmp 檔案和價格設定
   - 腳本會依序檢查背包每個格子
   - 如果格子內的物品符合 BMP 圖片且有價格設定:
     * 執行 Alt+右鍵上架
     * 自動輸入對應價格
     * 按下 Enter 確認
   - 符合 BMP 但無價格設定的物品會跳過不上架
   - 不符合的格子會自動跳過

【注意事項】

- 圖片識別容錯度設定為 *50 (可在程式碼中調整)
- 建議物品圖片清晰且背景單純
- 如果識別不準確,可以嘗試重新截圖或調整容錯度
- 支援多張圖片同時使用
- ⚠️ 重要: 只有在 prices.ini 中設定價格的物品才會上架
- 沒有價格設定的物品會被跳過,不會執行上架動作
- 腳本使用動態延遲機制,會自動調整速度以確保穩定性

【範例檔名】

混沌石.bmp
崇高石.bmp
神聖石.bmp
點金石.bmp
改造石.bmp
